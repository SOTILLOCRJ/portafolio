# Projetos com Bootstrap

Este diretório contém layouts e componentes criados com **Bootstrap**.

## Estrutura
- **/templates** → Páginas HTML com Bootstrap
- **/components** → Botões, cards, navbars e outros elementos
- **/themes** → Customizações de estilo

## Objetivo
Demonstrar habilidades em front-end com:
- Criação de interfaces responsivas
- Uso de componentes prontos do Bootstrap
- Customização de estilos para portfólio
