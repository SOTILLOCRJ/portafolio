# Projetos em PHP

Este diretório contém scripts e projetos desenvolvidos em **PHP** puro.

## Estrutura
- **/scripts** → Exemplos simples e utilitários
- **/projects** → Projetos completos em PHP
- **/examples** → Exercícios e práticas

## Objetivo
Mostrar domínio da linguagem PHP em:
- Manipulação de dados
- Programação orientada a objetos
- Criação de aplicações web sem frameworks
