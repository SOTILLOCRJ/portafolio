# Projetos em Laravel

Este diretório contém projetos e exemplos desenvolvidos com o framework **Laravel**.

## Estrutura
- **/app** → Lógica de negócio e controladores
- **/resources/views** → Templates Blade para o portfólio
- **/routes** → Definição de rotas
- **/public** → Arquivos públicos (CSS, JS, imagens)
- **/database** → Migrações e seeds

## Funcionalidades Demonstradas
- **API REST** → Criação de endpoints para comunicação entre cliente e servidor
- **Policies** → Controle de autorização baseado em regras de negócio
- **Gates** → Verificações simples de permissões para ações específicas
- **Autenticação** → Sistema de login e registro integrado
- **Middleware** → Proteção de rotas e manipulação de requisições

## Objetivo
Demonstrar habilidades em desenvolvimento back-end com Laravel, incluindo:
- Criação de APIs RESTful
- Implementação de segurança e autorização com Policies e Gates
- Integração com banco de dados e migrations
- Uso de Blade para renderização de páginas
