Este repositório reúne projetos de prática e estudo em **Laravel**, **PHP**, **Bootstrap**, **Docker** e **GitHub**, desenvolvidos para aprimorar lógica de programação, boas práticas e organização de código.  
Cada projeto possui seu próprio README detalhado com tecnologias utilizadas, objetivos, processo de desenvolvimento e instruções de execução.

---


## Objetivo do Portafolio
Este portafolio foi criado para servir como **vitrine profissional**, permitindo que recrutadores e colegas avaliem tanto o código quanto a organização do desenvolvimento.  
Aqui você encontrará práticas de programação, uso de Docker, versionamento com GitHub e aplicação de boas práticas em projetos reais.
